# 画面仕様書（就活コンパス フロントエンド）

作成：フロントエンドエンジニア ／ 2026-06-12

## 起動方法
```bash
cd フロントエンドエンジニア/成果物
npm install
npm run dev      # http://localhost:5173
```
※ 先にバックエンド（localhost:3001）を起動しておくこと。
`/api` へのリクエストはViteのプロキシで自動的に3001へ転送される。

## 画面構成
ヘッダーのナビゲーションで3つのビューを切替（ルーター不使用・状態管理のみ）。

### 1. ダッシュボード（Dashboard.tsx）
- 選考状況サマリー：ステータス別の企業件数をグリッド表示。
- 今後7日以内の締切：企業名（クリックで詳細へ）・ステータス・締切日。
- 今後7日以内の予定：未完了イベントを企業名・タイトル・日付で表示。

### 2. 企業一覧（CompanyList.tsx）
- テーブル表示（企業名／業界／ステータス／志望度★／締切／予定件数）。
- ステータスでのフィルタ（チップ型ボタン）。
- 「＋企業を追加」でインラインの登録フォームを開閉。
- 行クリックで詳細へ遷移。

### 3. 企業詳細・編集（CompanyDetail.tsx）
- 企業情報の表示／編集（編集ボタンでフォーム化）。
- 削除（確認ダイアログ付き）。
- 選考イベントの一覧・追加・完了チェック・削除。

## 設計メモ
- API通信は `src/api.ts` に集約。エラーはメッセージを画面上部に赤背景で表示。
- 型は `src/types.ts` でバックエンドと対応（Status enum等）。
- 外部UIライブラリ不使用（軽量・無料の方針）。スタイルは `styles.css` のみ。
