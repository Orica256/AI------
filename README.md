# 就活コンパス（Shukatsu Compass）

就職活動の **応募先・選考状況・締切** を一元管理するWebアプリです。
完全無料・ローカル完結（有料サービス不使用）。

## 主な機能
- 応募企業の登録 / 一覧 / 編集 / 削除（CRUD）
- 選考ステータス管理（気になる → エントリー → ES提出 → 面接 → 内定 / お祈り）
- 締切・面接日程の管理（残日数バッジで色分け表示）
- ダッシュボード（今週の締切、選考状況サマリー）
- カンバン表示（選考段階をドラッグ＆ドロップで移動）
- カレンダー表示（選考イベント・締切の月表示）
- 企業ごとのToDo（やること）管理
- メモ機能
- データのエクスポート / インポート（ローカルJSONでバックアップ・復元）

## 技術スタック
- フロント：React + TypeScript + Vite
- バック ：Node.js + Express + SQLite（better-sqlite3）

## 起動方法
詳細は各仕様書（`app/backend/API仕様書.md`・`app/frontend/画面仕様書.md`）を参照。

```bash
# 1) バックエンド起動（ターミナル1）
cd app/backend
npm install
npm run dev        # http://localhost:3001

# 2) フロントエンド起動（ターミナル2）
cd app/frontend
npm install
npm run dev        # http://localhost:5173
```

## ディレクトリ構成
```
就活コンパス/
├── CLAUDE.md                 … プロジェクト全体ルール
├── 作業ログ.md                … 全メンバーの作業ログ索引
├── app/                       … アプリ本体（動くコードはここが正）
│   ├── backend/               … API（Express + better-sqlite3）
│   └── frontend/              … UI（React + TypeScript + Vite）
├── PM/                        … 企画書・スケジュール・指示書・承認記録
├── フロントエンドエンジニア/     … フロントの作業記録（作業ログ・参考資料）
├── バックエンドエンジニア/       … バックの作業記録（作業ログ・参考資料）
├── レビュアー/                 … レビュー報告書・品質保証記録
└── 秘書/                       … 整理したユーザー意見・議事録
```
各メンバーフォルダは `参考資料/`・`成果物/`・`CLAUDE.md`・`作業ログ.md` を持ちます（動くコードは `app/` が正）。
