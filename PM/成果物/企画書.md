# 企画書：就活コンパス（Shukatsu Compass）

作成日：2026-06-12 ／ 作成者：PM

## 1. 背景・課題
就活を始めたばかりの学生は、複数企業の選考を同時並行で進めるため
「どの企業がどの段階か」「次の締切・面接がいつか」を見失いやすい。
スプレッドシート管理は煩雑で締切の見落としが起きやすい。

## 2. 解決策
応募企業・選考状況・締切を一元管理し、ダッシュボードで
「今やるべきこと」を即座に把握できるWebアプリを提供する。

## 3. ターゲット
就活中の大学生（まず作者本人が日常的に使えること）。

## 4. 提供価値
- 締切・面接の取りこぼし防止
- 選考状況の可視化による戦略立案
- ポートフォリオとしての提示（フルスタック開発実績）

## 5. スコープ（MVP）
- [x] 応募企業のCRUD（登録・一覧・編集・削除）
- [x] 選考ステータス管理
- [x] 締切・面接日程の管理（企業ごとのイベント）
- [x] ダッシュボード（今週の締切、ステータス別件数）
- [x] メモ

## 6. スコープ外（今回やらない）
- ユーザー認証・複数ユーザー（ローカル単独利用のため不要）
- クラウド同期・通知メール（有料・外部依存になるため）
- スマホネイティブ対応

## 7. 制約（最重要）
- **完全無料**：有料API・有料クラウド・課金サービス禁止。
- **ローカル完結**：データは手元のSQLiteファイルのみ。外部送信なし。
- 誰でも `npm install` → 起動コマンドで動くこと。

## 8. 成功基準
1. バックエンド・フロントエンドが起動し、企業の登録〜削除が一通りできる。
2. ダッシュボードに今週の締切と状況サマリーが出る。
3. レビュアーの品質チェックに合格し、PMが承認する。
