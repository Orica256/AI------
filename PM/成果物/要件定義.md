# 要件定義：就活コンパス

作成日：2026-06-12 ／ 作成者：PM

## 1. データモデル
### companies（応募企業）
| カラム | 型 | 説明 |
|--------|----|----|
| id | INTEGER PK | 自動採番 |
| name | TEXT NOT NULL | 企業名 |
| industry | TEXT | 業界 |
| status | TEXT NOT NULL | 選考ステータス（下記enum） |
| priority | INTEGER | 志望度 1〜5 |
| applied_date | TEXT | 応募日（YYYY-MM-DD） |
| deadline | TEXT | 直近の締切（YYYY-MM-DD） |
| memo | TEXT | メモ |
| created_at | TEXT | 作成日時 |
| updated_at | TEXT | 更新日時 |

### events（選考イベント）
| カラム | 型 | 説明 |
|--------|----|----|
| id | INTEGER PK | 自動採番 |
| company_id | INTEGER FK | 企業ID |
| title | TEXT NOT NULL | イベント名（例：一次面接） |
| date | TEXT NOT NULL | 日付（YYYY-MM-DD） |
| done | INTEGER | 完了フラグ 0/1 |

## 2. 選考ステータス（enum）
`気になる` / `エントリー` / `ES提出` / `一次面接` / `二次面接` / `最終面接` / `内定` / `お祈り`

## 3. API仕様（REST）
| メソッド | パス | 説明 |
|----------|------|------|
| GET | /api/companies | 企業一覧（イベント件数付き） |
| GET | /api/companies/:id | 企業詳細（イベント含む） |
| POST | /api/companies | 企業作成 |
| PUT | /api/companies/:id | 企業更新 |
| DELETE | /api/companies/:id | 企業削除 |
| POST | /api/companies/:id/events | イベント追加 |
| PUT | /api/events/:id | イベント更新（完了切替等） |
| DELETE | /api/events/:id | イベント削除 |
| GET | /api/dashboard | サマリー（ステータス別件数・今後7日の締切/イベント） |

## 4. 画面要件
1. **ダッシュボード**：ステータス別件数、今後7日以内の締切・イベント一覧。
2. **企業一覧**：テーブル表示、ステータスでフィルタ、新規追加ボタン。
3. **企業詳細/編集**：フォームで編集、イベントの追加・完了切替・削除。

## 5. 非機能要件
- 無料・ローカル完結（制約遵守）。
- TypeScript strict。
- 起動手順をREADMEに明記。
- 入力バリデーション（必須項目・日付形式）。
