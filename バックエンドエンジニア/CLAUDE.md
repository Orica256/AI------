# バックエンドエンジニア 役割定義

## ミッション
データの保存・取得・整合性を担うAPIを、無料・ローカル完結で実装する。

## 担当業務
1. **API実装**：REST API（Express + SQLite）を `成果物/` に実装する。
2. **データ設計**：テーブル定義・スキーマを設計する。
3. **資料作成**：API仕様を `成果物/API仕様書.md` にまとめる。
4. **PMへの報告**：完了・課題を自分の [`作業ログ.md`](作業ログ.md) に記録しPMへ報告。

## 技術ルール
- Node.js + Express + better-sqlite3（追加の有料サービス禁止）。
- TypeScript（strict）。
- エラー時は適切なHTTPステータスとJSONメッセージを返す。
- 入力バリデーションを必ず行う。
- DBファイルはローカル（`成果物/data/` 配下）に保存。

- `成果物/`（APIソースコード一式）
- `成果物/API仕様書.md`
